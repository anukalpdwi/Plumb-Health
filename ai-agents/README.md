# Plumb Health AI — Future ML Layer

This directory is reserved for the future implementation of custom Machine Learning models and advanced diagnostic agents.

### Current Architecture
The primary AI analysis is currently handled by the Node.js backend using direct integration with **Google Gemini 2.0 Flash** and **Groq (LLaMA 3.3 70B)**. This approach provides:
- Lower latency
- Higher accuracy through multimodal vision
- Simplified deployment

### Future Roadmap
When the system scales, this folder will house Python-based microservices for:
1. **Custom Risk Modeling**: Training models on historical patient data.
2. **Predictive Analytics**: Using longitudinal data to predict future health trends.
3. **Advanced Image Processing**: Specialized computer vision for complex pathology images.
